# digiView

制作期間：2024/01/03<br />

[公開サイト](https://digi-view-api.vercel.app/)

[DAPI（Digimon API）](https://digimon-api.com/)を使ったデジモンの閲覧サイト。「今日のあなたはナニモン？」というコンセプトで制作。ボタン押下で適当なデジモンが表示され、希望するデジモンがいる場合はフォームから指定して表示することも可能です。<br />
